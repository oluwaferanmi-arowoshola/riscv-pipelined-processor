# Project Overview

This project implements a five-stage pipelined RISC-V processor using SystemVerilog.

The goal of the design was to understand and implement key architectural concepts including:

- pipeline execution
- hazard detection
- data forwarding
- instruction verification
- simulation-based validation

The processor architecture follows the classical IF–ID–EX–MEM–WB pipeline structure and was verified using simulation in Vivado.

Key learning outcomes from the project include:

- designing and debugging pipeline datapaths
- implementing hazard detection logic
- verifying instruction correctness through simulation
- analyzing pipeline timing and instruction overlap
